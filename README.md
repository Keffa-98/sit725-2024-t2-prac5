# Week 5 : SIT 725 Prac 5 MVC Architecture

## Installation 
clone the repository
```
git clone https://github.com/Keffa-98/sit725-2024-t2-prac5
```

## Install dependencies

change directory to sit725-2024-t2-prac5

```
cd sit725-2024-t2-prac5
```

install node_modules via npm 

`npm install
`

## Run the application

use the following command to run the application
```
npm run start
```

## Test Application

In your favourite browser navigate to https://localhost:3000
